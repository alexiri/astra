How To Use Images in Email Templates

Upload images for use in email templates.
You can reference an image in two ways:
  • Inline (embedded) image: <img src="{% templatetag openblock %} inline_image '{{ mail_images_external_example_url }}' {% templatetag closeblock %}" /> (requires {% templatetag openblock %} load post_office {% templatetag closeblock %} in the template; when sending, the app embeds the image by reading it from storage, so the URL does not need to be reachable from the Django server via HTTP)
  • External URL: <img src="{{ mail_images_external_example_url }}" />
Notes: previews rewrite {% templatetag openblock %} inline_image ... {% templatetag closeblock %} to a plain URL so your browser can display it. Plain-text previews drop images and any {% templatetag openblock %} ... {% templatetag closeblock %} tags.
